# gesture-control
Gesture-controlled desktop system inspired by Iron Man-style UIs, built using Python, MediaPipe, OpenCV, pyautogui, and pywinauto. Uses real-time hand tracking and pinch gestures to move windows across screens locally. Built through rapid prototyping, experimentation, and vibe coding.

SET-UP GUIDE:-
1. install  zip||
2.extract in a empty folder||
3.open cmd for the folder path||
4.you have to install required things in requirement.txt
  use command:- py -m pip install -r requirements.txt
||
5.once everything is installed we need an environment 
 use command:- py -m venv gesture-env
               gesture-env\Scripts\activate       
||
6.finally run using:-
  py main.py(for windows)||
